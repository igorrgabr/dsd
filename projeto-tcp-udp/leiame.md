## Projeto TCP/UDP
### Instalação
- python -m venv projeto
- projeto\Scripts\activate
- cd projeto
- git clone https://github.com/igorrgabr/dsd.git
- cd dsd/projeto-tcp-udp
- pip install -r requeriments.txt
